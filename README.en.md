# OCE SQL Teaching Materials

中文版: [README.md](README.md)

This repository contains course materials for **Structured Query Language (SQL)**.
It is designed for students to learn SQL step by step through weekly practices and syllabus-guided topics.

## Course Focus

- SQL fundamentals and query writing
- Multi-table queries, subqueries, and set operators
- Data manipulation (`INSERT`, `UPDATE`, `DELETE`)
- Schema objects and basic database administration concepts

## Repository Structure

- `syllabuses/`
  - Course syllabus documents (e.g., weekly schedule and grading policy)
- `course_practices/`
  - Practice handouts by workshop/unit (e.g., `ws1-u01_en.md`, `ws2-u06_en.md`)

## Syllabus Links

- Chinese syllabus: [syllabuses/syllabus_114B_ch.md](syllabuses/syllabus_114B_ch.md)
- English syllabus: [syllabuses/syllabus_114B_en.md](syllabuses/syllabus_114B_en.md)

## How To Use This Repository (for Students)

1. Start with the syllabus in `syllabuses/` to understand weekly goals.
2. Follow practice files in `course_practices/` in unit order (`u01`, `u02`, ...).
