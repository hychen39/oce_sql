# OCE SQL 教材

English version: [README.en.md](README.en.md)

本倉庫為 **SQL（Structured Query Language）** 課程教材，提供學生依照週次與單元逐步學習。

## 課程重點

- SQL 基礎與查詢語法
- 多表查詢、子查詢、集合運算
- 資料維護（`INSERT`、`UPDATE`、`DELETE`）
- 資料庫物件與基本管理概念

## 目錄結構

- `syllabuses/`
  - 課程大綱與週次安排
- `course_practices/`
  - 各單元練習教材（例如 `ws1-u01_en.md`、`ws2-u06_en.md`）

## 學生使用方式

1. 先閱讀 `syllabuses/` 中的大綱，掌握每週學習目標。
2. 依單元順序完成 `course_practices/` 中的教材內容。

## 備註

- 部分教學內容可能參考 Oracle 著作權教材。
- 本倉庫內容以教學與學習用途為主。
